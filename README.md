Road Accident Analysis in India - Power BI Project
This project focuses on analyzing road accident data to identify patterns and causes. I used Power BI to create an interactive dashboard that helps in understanding the factors contributing to road fatalities across the country.
Dashboard Preview
Key Insights from the Analysis
Major Causes: Identified "No distancing" and "Improper lane changes" as the primary behavioral reasons for accidents.
Age Group Trends: Data shows the 18-35 age group is the most affected demographic in road incidents.
Road Conditions: Nearly 92% of accidents occur on Asphalt roads, indicating high risks on major infrastructure.
Vehicle Ownership: More than 82% of involved vehicles were driven by employees rather than the owners.
Technical Contributions
Data Cleaning: Structured and cleaned raw datasets using MS Excel and Power Query for accurate analysis.
Dashboard Design: Developed an interactive layout with dynamic slicers for Weather, Lighting, and Gender.
Documentation: Prepared a detailed analytical report and a summary presentation based on the findings.
